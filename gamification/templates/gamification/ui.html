🎮 Gamification API

Sistema de Recompensas e Pontos

ℹ️ Informações: Esta é uma interface de documentação. Para usar a API, faça requisições aos endpoints abaixo usando Token Authentication ou Basic Auth.

Recompensas

  • GET Listar Recompensas Lista paginada de todas as recompensas disponíveis
  • POST Criar Recompensa Criar uma nova recompensa (autenticação necessária)

Recompensas do Usuário

  • GET Listar Minhas Recompensas Recompensas conquistadas pelo usuário autenticado
  • GET Recompensas por Usuário Filtrar recompensas de um usuário específico

Autenticação

Para acessar endpoints protegidos, use um dos seguintes métodos:

  • Token Authentication:

    Authorization: Token seu_token_aqui

  • Basic Authentication:

    Authorization: Basic base64(username:password)

  • Session Authentication:

    Disponível quando acessado via navegador com usuário logado
