🖨️ Printing API

Gerenciamento de Impressoras e Filas de Impressão

ℹ️ Informações: Esta é uma interface de documentação. Para usar a API, faça requisições aos endpoints abaixo usando Token Authentication ou Basic Auth.

Impressoras

  • GET Listar Impressoras Lista paginada de todas as impressoras cadastradas
  • POST Criar Impressora Registrar uma nova impressora no sistema

Impressoras Ativas

  • GET Listar Impressoras Ativas Apenas impressoras em status ativo

Gerenciamento de Status

  • POST Marcar Manutenção Rota: /printing/impressoras/{id}/marcar_manutencao/ - Mudar status para manutenção
  • POST Ativar Impressora Rota: /printing/impressoras/{id}/ativar/ - Restaurar status para ativo

Autenticação

Para acessar endpoints, use um dos seguintes métodos:

  • Token Authentication:

    Authorization: Token seu_token_aqui

  • Basic Authentication:

    Authorization: Basic base64(username:password)

  • Session Authentication:

    Disponível quando acessado via navegador com usuário logado
