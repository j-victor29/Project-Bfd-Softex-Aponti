📦 Orders API

Gerenciamento de Pedidos e Itens

ℹ️ Informações: Esta é uma interface de documentação. Para usar a API, faça requisições aos endpoints abaixo usando Token Authentication ou Basic Auth.

Pedidos

  • GET Listar Pedidos Lista paginada de todos os pedidos
  • POST Criar Pedido Criar um novo pedido (autenticação necessária)

Operações em Pedidos

  • POST Adicionar Item Rota: /orders/orders/{id}/items/ - Adicionar artigo ao pedido
  • POST Finalizar Pedido Rota: /orders/orders/{id}/finalizar/ - Completar o pedido
  • POST Marcar como Enviado Rota: /orders/orders/{id}/marcar-enviado/ - Atualizar status para enviado
  • POST Marcar como Impresso Rota: /orders/orders/{id}/marcar-impresso/ - Registrar impressão
  • GET Status do Pedido Rota: /orders/orders/{id}/status/ - Consultar status atual

Autenticação

Para acessar endpoints, use um dos seguintes métodos:

  • Token Authentication:

    Authorization: Token seu_token_aqui

  • Basic Authentication:

    Authorization: Basic base64(username:password)

  • Session Authentication:

    Disponível quando acessado via navegador com usuário logado
